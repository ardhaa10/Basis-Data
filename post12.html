Post 12: Apa itu SQL, DDL, DML, DCL, dan TCL?

Dipublikasikan: 1 November 2025

Dalam dunia basis data, kita mengenal istilah SQL sebagai bahasa standar untuk mengelola dan memanipulasi data di sistem manajemen basis data relasional (RDBMS). SQL dibagi menjadi beberapa bagian utama berdasarkan fungsinya, yaitu DDL, DML, DCL, dan TCL.

1. SQL (Structured Query Language)

SQL adalah bahasa untuk membuat, mengubah, dan mengelola basis data. Dengan SQL, kita bisa membuat tabel, mengisi data, menampilkan, memperbarui, atau menghapus data dengan perintah yang sederhana.

2. DDL (Data Definition Language)

Digunakan untuk mendefinisikan struktur database seperti tabel, kolom, dan relasi.
Contoh perintah: CREATE, ALTER, DROP.

3. DML (Data Manipulation Language)

Digunakan untuk memanipulasi data di dalam tabel.
Contoh perintah: SELECT, INSERT, UPDATE, DELETE.

4. DCL (Data Control Language)

Digunakan untuk mengatur hak akses pengguna terhadap data.
Contoh perintah: GRANT, REVOKE.

5. TCL (Transaction Control Language)

Berfungsi untuk mengatur transaksi dalam database, memastikan data tetap konsisten.
Contoh perintah: COMMIT, ROLLBACK, SAVEPOINT.

Contoh sederhana:
BEGIN;
INSERT INTO pelanggan VALUES (1, 'Andi', 'Bandung');
COMMIT;

Kesimpulan: SQL adalah bahasa utama untuk berinteraksi dengan basis data. DDL mendefinisikan struktur, DML mengelola data, DCL mengatur hak akses, dan TCL memastikan transaksi berjalan konsisten.
