Post 5: Installasi MySQL

Apa Itu MySQL?

MySQL adalah sistem manajemen basis data relasional (RDBMS) yang menggunakan bahasa SQL (Structured Query Language). MySQL banyak digunakan untuk aplikasi web, sistem informasi, hingga platform besar karena:

  • Gratis dan open-source
  • Cepat dan stabil
  • Mendukung banyak bahasa pemrograman
  • Bisa berjalan di Windows, Linux, dan macOS
  • Banyak digunakan bersama PHP dan Apache (seperti pada XAMPP)

MySQL menyimpan data dalam bentuk tabel dan memungkinkan pengguna melakukan operasi seperti menyimpan, mengubah, menghapus, dan mengambil data.

Langkah-Langkah Instalasi MySQL di Windows

  1. Download Installer
    • Buka situs resmi: https://dev.mysql.com/downloads/installer/
    • Pilih MySQL Installer (Windows).
    • Unduh versi MySQL Installer Full agar lengkap.
  2. Jalankan Installer
    • Klik dua kali file .msi yang sudah diunduh.
    • Pilih opsi instalasi: Developer Default (direkomendasikan) atau Server only jika hanya butuh database.
  3. Pilih Komponen
    • MySQL Server
    • MySQL Workbench
    • MySQL Shell
    • Connector (jika dibutuhkan)
  4. Proses Instalasi
    • Klik Execute atau Install untuk memulai proses pengunduhan dan pemasangan.
  5. Konfigurasi Server
    • Pilih tipe konfigurasi: Standalone
    • Pilih port default: 3306
    • Pilih mode autentikasi: Use Strong Password Encryption
  6. Buat Password
    • Buat password untuk user root.
    • Opsi untuk menambah user lain (bisa dilewati dulu).
  7. Selesaikan Setup
    • Klik Apply dan tunggu proses konfigurasi selesai.
    • Klik Finish.
  8. Coba Jalankan
    • Gunakan MySQL Workbench dan login dengan:
    • User: root
    • Password: (yang sudah dibuat sebelumnya)
